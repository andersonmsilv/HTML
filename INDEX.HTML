Estrutura básica do HTML

Uma pagina HTML é composta por três coisas:

  • No inicio<"HTML"> e no final de tudo<"/HTML">
  • <"HEAD">cabeçalho <"/HEAD">
  • <"BODY">corpo em que aparece para o usuario <"/BODY">

Dessa maneira ficaria:

< HTML>
< HEAD> < TITLE> "Titulo da pagina" < /TITLE> < /HEAD>
< BODY> Aqui é onde vai todas as informações que serão carregadas para o usúario<"/BODY">
<"/HTML">
